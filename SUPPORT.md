# Support

Loci is pre-1.0.

## Questions

Use GitHub Discussions if enabled. If not, open a normal issue with the `question` label.

## Bugs

Open a bug report and include:

- macOS version
- Loci version or commit
- install method
- library size
- steps to reproduce

Do not include secrets, OAuth tokens, API keys, private bookmarks, prompts, file contents, or local library data.

## Security Or Privacy Issues

Read [SECURITY.md](SECURITY.md). Do not post sensitive vulnerability details publicly.
