# Code of Conduct

Loci should be a practical, respectful project for people building local-first creative tools.

## Expected Behavior

- Be direct and kind.
- Assume good intent, but fix real problems clearly.
- Keep discussions focused on the work.
- Respect user privacy and local data ownership.
- Give actionable feedback.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private data, credentials, tokens, or user libraries.
- Repeatedly derailing technical discussions.
- Pressuring maintainers to ship unsafe privacy or security behavior.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or contributors that make the project unsafe or unproductive.
